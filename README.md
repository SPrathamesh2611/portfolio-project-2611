## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Go to project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
